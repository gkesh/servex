# Servex - Service Exhange

## Description

Servex is an idea for an online platform where people can exhange
thier skilled or semi-skilled labor for money.

For example
- Tuitions for Unemployed Students
- Baby Sitting for Busy Parents
- and so on...

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
